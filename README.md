# Savio

**Savio** — это кроссплатформенное десктопное приложение для скачивания видео и аудио с популярных онлайн-платформ. Приложение позволяет быстро загружать контент по ссылке, автоматически выбирая максимально доступное качество.

## ✨ Возможности

- 🖥️ Кроссплатформенное приложение (Windows, Linux и macOS).
- 📦 Готовые сборки для всех трёх систем — устанавливать ничего не нужно.
- ⚙️ Автоматическая доустановка `yt-dlp` и `ffmpeg` при первом запуске:
  без прав администратора, без окна терминала, с прогрессом в окне программы.
- 🎥 Скачивание видео в формате **MP4**.
- 🎵 Скачивание аудио в формате **MP3**.
- 🏆 Автоматическая загрузка **MP4** в максимально доступном разрешении.
- 🎧 Автоматическая загрузка **MP3** в максимально доступном качестве.
- ⚡ Быстрая обработка ссылок и загрузка файлов.
- 📁 Сохранение файлов в выбранную пользователем папку.
- 🌐 Поддержка нескольких видеоплатформ.
- 🔌 Простая архитектура, позволяющая легко добавлять новые источники.
- 🌙 Тёмный интерфейс с жёлтыми акцентами, читаемый при долгой работе.
- 📊 Прогресс загрузки: проценты, объём, скорость и оставшееся время.
- ⏹️ Отмена загрузки в любой момент.
- 🔄 Обновление движка `yt-dlp` одной кнопкой — прямо из окна программы.
- 🧹 Просмотр и удаление метаданных (EXIF, ID3, XMP, IPTC) у файлов на диске:
  геометка, модель камеры, дата съёмки, теги и обложка — без пересжатия
  и без потери качества.

## 📥 Установка

Скачайте файл для своей системы со страницы
[Releases](https://github.com/cholakhyanerik/savio/releases):

| Система | Файл |
|---|---|
| Windows | `savio-windows-x86_64.exe` |
| macOS (Intel и Apple Silicon) | `savio-macos-universal.dmg` |
| Linux | `savio-linux-x86_64.AppImage` или `savio-linux-x86_64` |

Установщик не нужен: это готовые к запуску файлы.

**macOS.** Приложение не подписано сертификатом Apple Developer, поэтому при
первом открытии система сообщит, что не может проверить разработчика.
Обойти это можно так: **Системные настройки → Конфиденциальность и
безопасность**, найти сообщение о Savio и нажать **«Всё равно открыть»**.

**Linux.** AppImage нужно сделать исполняемым: `chmod +x savio-*.AppImage`.
Сборка сделана на Ubuntu 24.04 и требует glibc не ниже 2.39 — на более старых
дистрибутивах (Ubuntu 22.04, Debian 12) она не запустится, там придётся
собрать Savio из исходников.

**Windows.** Файл не подписан, поэтому SmartScreen может показать синее окно
«Windows защитила ваш компьютер» — нажмите **«Подробнее» → «Выполнить в любом
случае»**.

## ⚙️ Требования

Savio — графическая обёртка над двумя внешними программами:

- **`yt-dlp`** — обязателен, скачивает сам ролик.
- **`ffmpeg`** (вместе с `ffprobe`) — склеивает видео со звуком (**MP4**) и
  извлекает звук (**MP3**).

**Ставить их вручную не нужно.** При первом запуске Savio проверяет, есть ли
они на компьютере, и, если чего-то не хватает, догружает недостающее сам —
без прав администратора и без окна терминала, показывая окно с прогрессом.
Загружается около 300 МБ, дальше запуск мгновенный. Если всё уже установлено,
проверка проходит незаметно.

Инструменты кладутся в папку пользователя и больше никуда:

| Система | Куда |
|---|---|
| Windows | `%LOCALAPPDATA%\Savio\bin` |
| macOS | `~/Library/Application Support/Savio/bin` |
| Linux | `~/.local/share/savio/bin` |

Ищет их Savio в трёх местах по порядку: рядом со своим исполняемым файлом
(так можно носить всё одной папкой), затем в системном `PATH`, затем в папке
выше. Уже установленный в системе `ffmpeg` всегда в приоритете над скачанным.

Если установка не удалась — например, нет интернета, — Savio всё равно
откроется и подскажет, что делать. Без `ffmpeg` программа работает, но
предупреждает: склейки и конвертации не будет.

> Антивирус иногда принимает `yt-dlp.exe` за угрозу — это известное ложное
> срабатывание на программы, собранные PyInstaller. Если файл пропал сразу
> после загрузки, добавьте папку из таблицы выше в исключения антивируса
> либо положите `yt-dlp.exe` рядом с `savio.exe` вручную.

## 🌍 Поддерживаемые платформы

Savio скачивает всё, что умеет `yt-dlp`, — это больше полутора тысяч сайтов,
среди них:

- YouTube
- VK Видео
- Rutube, Dailymotion, Vimeo, SoundCloud и другие

Полный и всегда актуальный список ведёт сам `yt-dlp`:
[supportedsites.md](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md).
Своего списка Savio не хранит намеренно: он менялся бы с каждым выпуском
`yt-dlp` и устаревал бы молча.

Если сайта в списке нет, Savio так и скажет — вместо непонятной ошибки. А если
сайт там есть, но ссылка не скачивается, обычно помогает кнопка
**«Обновить движок»** (см. ниже).

## 🚀 Использование

Окно разделено на две вкладки: **«Загрузка»** — скачивание по ссылке,
**«Метаданные»** — работа с файлами, которые уже лежат на диске.

### Загрузка

0. При самом первом запуске дождитесь окончания установки — окно с прогрессом
   закроется само. Дальше этот шаг больше не повторится.
1. Вставьте ссылку на видео.
2. Выберите формат:
   - **MP4** — видео в максимально доступном разрешении.
   - **MP3** — аудио в максимально доступном качестве.
3. Выберите папку для сохранения (при необходимости).
4. Нажмите кнопку **«Скачать»**.
5. Дождитесь завершения загрузки.

### 🧹 Просмотр и удаление метаданных

Фотографии и музыкальные файлы носят в себе больше, чем видно глазом: модель
камеры, дату съёмки, **координаты места**, имя автора, обложку альбома. Вкладка
**«Метаданные»** позволяет посмотреть на это и стереть перед тем, как отправить
файл кому-то ещё.

1. Откройте вкладку **«Метаданные»** и нажмите **«Выбрать файл…»**.
2. **«Читать»** покажет список всего, что записано в файле.
3. **«Удалить»** сотрёт теги, геометку и обложку.

Поддерживаются **MP3** и изображения: **JPG**, **PNG**, **WebP**, **GIF**.
Для **TIFF** доступно только чтение. Видео пока не обрабатывается — при выборе
видеофайла кнопки останутся выключенными.

**Качество не меняется.** Savio не пересжимает ни картинку, ни звук: из файла
вырезаются только служебные блоки, а сами пиксели и звуковые кадры копируются
как есть. Поэтому очистка происходит мгновенно даже на больших файлах.

**Файл перезаписывается на месте**, копия рядом не создаётся — поэтому Savio
переспрашивает перед удалением. Вернуть стёртые метаданные будет нельзя.

### 🔄 Если ссылка перестала скачиваться

Сайты меняются, и `yt-dlp` со временем перестаёт их понимать. Внизу окна есть
кнопка **«Обновить движок»** — она скачивает свежий `yt-dlp` и сообщает, с
какой версии на какую обновилась (или что версия и так последняя).

Savio обновляет **только свою** копию — ту, что скачал сам при первом запуске.
Если `yt-dlp` установлен в системе (`winget`, `brew`, пакетный менеджер) или
лежит рядом с Savio в портативной поставке, программа не станет подменять
чужой файл, а покажет путь к нему и команду для обновления. Иначе Savio
разошёлся бы с пакетным менеджером, а обновление могло бы вообще не подействовать:
системная копия всё равно имеет приоритет над скачанной.
